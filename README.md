# AI Programming Assignment4

This repository contains the codes and documentations for the following questions:
```
1.Implement the Map Coloring problem using CSP for the seven principal states and territories of Australia namely: WA, NT, Queensland, SA, NSW, V, T.
2.Apply the Map coloring problem to the 33 districts of Telangana. A sample output of your code may look as shown. Choose your own colors.
3.Implement the Sudoku puzzle using the CSP. (Refer 5.2.6)
4.Implement the crypt-analysis puzzle in Fig. 5.2a
```

##  How to Run
### 1. Clone the Repository
```
git clone https://github.com/m-o-n-i-sh/se24ucse153_AI_programing_assignment_4.git
cd se24ucse153_AI_programing_assignment_4.git
```
### 2. Run Individual File
Each question is contained within its own folder. To run a specific question,navigate into its folder and execute the Python script,example:
```
cd Q1_Map_Coloring_Australia
python map_coloring_aus.py
```
